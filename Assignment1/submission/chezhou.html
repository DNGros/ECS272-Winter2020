Image Source: A global wildfire dataset for the analysis of fire regimes and fire behaviour

This image is a static visualization of global "fire density", which displays the number of fires per region surface from 2001 to 2017. The warm color, such as red and yellow, indicates a larger number of fires compare to the cold color like green and blue.

In my opinion, this image is a good visualization. Firstly, it clearly shows the relationship between fire density and geographical location. Secondly, the visualization utilizes warm color to highlight regions with higher fire density, which is both direct and intuitive. These techniques make this visualization easy to understand.
