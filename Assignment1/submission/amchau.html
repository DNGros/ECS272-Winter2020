Description

This visualization is showing the softmax prediction of three neural digit recognition models as the character is being drawn. We can see how the models are gravitating towards their predictions differently giving the same input.

This visualization is not interactive, but is animated. It is presented as a gif format and is supposed to be played passively.

In terms of colors, it uses different colors to differentiate the three ML model predictions for an easier view. The color saturation is proportional to the activation value, so that higher activation is more saturated. All softmax plots are bar plots and aligned with each other, such that it can be very easy to see the different activations across the ML models. The animation allows us to see the digits drawn out across time, which is relevant to what the visualization is trying to demonstrate.

This visualization is chosen since it is the most relevant way to demonstrate the models activation across time. The design is minimalistic, only showing the most important details (the activation value, and the digit), with colors and layout designed for easy viewing. Bar plot is chosen as it can show the proportion of activation values clearly.

I think this is a good visualization as it gets its point across very efficiently. The main thing to be displayed is the change of activation values across time, which we can see clearly with the animation and the change in color saturation. The layout also allows us to compare the ML models easily. We can also see how the activation changes as the digits is being drawn, allowing us to easily see what the ML model thinks as the input digit is realized. The design is also minimalistic and clean, not getting in the way of the data it is trying to present.
